  • Qu'est-ce que la Code Game JAM ?
  • Site officiel de la Code Game JAM
  • placeContexte
    La Code Game Jam est un concours de programmation organisé par le département Informatique de l'IUT de Montpellier. Les équipes, composée de 4 à 6 étudiants du département, ont 30 heures pour réaliser un jeu vidéo en respectant un thème spécifique. Le support, les outils, les langages utilisés sont libres. Le thème de la troisième édition était "Ballade en forêt"
  • groupÉquipe et organisation
    Notre équipe était composée de 6 étudiants de première année. Nous faisions tous partie du même groupe de TD. Nous nous sommes accordés sur la création d'un jeu de type platformer. Nous avons utilisés le moteur de jeu Unity, le langage utilisé le C#. Les éléments graphiques proviennent d'un Asset d'Unity et la musique a été composée et réalisée sur place par des membres de l'équipe.
  • whatshotRéalisation personnelle
    Je me suis chargé de la réalisation du niveau, la création des éléments graphiques du décor à partir de notre Asset graphique. J'ai réalisé la disposition du niveau, l'arrière plan et l'avant-plan.

Notre Dominion

  • flight_takeoffRéalisation
    Le but de ce projet était de produire une implémentation en Java du jeu de cartes Dominion. Dominion est un jeu de cartes inventé par Donald X. Vaccarino pour 2 à 4 joueurs. C'est un jeu de construction de deck où les joueurs doivent à tour de rôle jouer des cartes de leur main pour obtenir de nouvelles cartes parmi un ensemble commun disponible. Lorsque la partie s'arrête le joueur possédant le plus de points de victoire dans son deck est déclaré vainqueur.
  • groupUn travail d'équipe
    Ce projet a été réalisé par la collaboration entre 4 étudiants de première année de DUT Informatique. La répartition du travail était équitable, et le projet s'est déroulé dans de bonnes conditions.
